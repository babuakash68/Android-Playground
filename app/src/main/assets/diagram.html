onCreate()

Component Interaction

Activities can:

  • Start Services
  • Register Broadcast Receivers
  • Access Content Providers

Services can:

  • Send Broadcasts
  • Access Content Providers

Broadcast Receivers can:

  • Start Services
  • Access Content Providers

Content Providers can:

  • Be accessed by all other components
  • Notify data changes to observers
